## How To Use
### 1. Visit the FarmCare platform.  
Start by capturing a clear image of a plant leaf that might be showing signs of disease. Ensure that the image is sharp and well-lit. Easily upload the image to the FarmCare platform using our user-friendlyÂ interface.
### 2. Upload an image of your plant.  
FarmCare's advanced AI technology kicks into action. The uploaded image is processed by a sophisticated machine learning model to analyze the visual characteristics of the leaf in the image, comparing them to an extensiveÂ database.
### 3. Wait for the diagnostic results and read about the disease description.
FarmCare provides you with comprehensive insights. It not only identifies the specific disease affecting your plant but also offers a detailed description of the plant, the disease itself, and recommends effectiveÂ remedies.

## Setup FarmCare Locally
Once you have followed the steps mentioned in [Contribution](https://github.com/rahulkarda/nextgencoders_hackhive/blob/master/Contributing.md) to fork and clone the repository, do the following to setup FarmCare locally on your PC.
```bash
cd web
```
Install all the dependencies
```bash
npm ci
```
Add the API key to the `.env` folder as mentioned in `.env.example`
Get the Plant Id API key from [here](https://web.plant.id/plant-identification-api/)
```
NEXT_PUBLIC_PLANT_ID_API_KEY = "Your API Key"
```

Run the server locally
```bash
npm run dev
```
That's it you are good to go! Looking forward to your contributions.
